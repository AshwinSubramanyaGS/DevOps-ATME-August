#### **1. Navigate to a Directory**

* **Task:** Go to the `/tmp` directory.

```bash
cd /tmp
```

---

#### **2. Create a New Directory**

* **Task:** Create a directory named `practice1`.

```bash
mkdir practice1
```

---

#### **3. Enter the New Directory**

* **Task:** Change into the `practice1` directory.

```bash
cd practice1
```

---

#### **4. Create Multiple Files**

* **Task:** Create 3 empty files: `file1.txt`, `file2.txt`, and `file3.txt`.

```bash
touch file1.txt file2.txt file3.txt
```

---

#### **5. Create a Subdirectory**

* **Task:** Create a subdirectory inside `practice1` called `archive`.

```bash
mkdir archive
```

---

#### **6. Move Files into the Subdirectory**

* **Task:** Move `file2.txt` and `file3.txt` into the `archive` directory.

```bash
mv file2.txt file3.txt archive/
```

---

#### **7. Rename a File**

* **Task:** Rename `file1.txt` to `main.txt`.

```bash
mv file1.txt main.txt
```

---

#### **8. Navigate Using Relative Paths**

* **Task:** From within `practice1`, go into the `archive` directory using a relative path.

```bash
cd archive
```

---

#### **9. Go Back to the Parent Directory**

* **Task:** Navigate back to `practice1` from `archive`.

```bash
cd ..
```

---

#### **10. Delete a File and Directory**

* **Task:** Delete `main.txt` and then delete the empty `archive` directory.

```bash
rm main.txt
rmdir archive
```
